put example:

Account_tool_character.php will produce an link to this tool with title:
"Account tool character".
text inside Account_tool_character.txt will create description message 
beneath the link in panel (this is optional)

HOW TO USE CUSTOM DESCR FILES (module_name.txt):
    <image.png>|<message>
    example with custom image:
    star.png|Donate to us using PayPal and earn Donation Points!
    example with default image (modules.png):
    |Donate to us using PayPal and earn Donation Points!
    
    
    Location of folder that script is looking for images:
    www.yourserver.com/styles/<current_active_style>/images/
